# Tool for Fundamental Aerodynamics Course

Python tools for 2D potential flow demos: analytical cylinder solution (with optional circulation), reusable source/vortex-panel utilities, and a panel-method solver for airfoils. 

## Quick start

```bash
python -m pip install -e .

# run a couple of demos
python examples/example_cylinder_flow.py
python examples/example_singularity_flow_fields.py
```
